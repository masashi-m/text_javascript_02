◆switch 文

switcht 文は、条件の値毎に実行処理を記述します。
値はcaseで場合分けし、一致した case に紐づく処理を言実行します。
以下、記述方法を記載します。

let plan = "B";

switch (plan) {
  case "A":
    console.log("travel to Waikiki");
    break;
  case "B":
    console.log("travel to Bali");
    break;
  case "C":
    console.log("travel to Naha");
    break;
  default:
    console.log("plan is nothing");
}

上記コードですが、まず switch の後ろに条件を記述し、条件の結果を case で場合分けします。
今回は変数 plan を設定し、plan の結果(値)によって場合分けしています。

case の後ろは条件式の値 + コロン、その後ろは実行処理を記述します。
実行処理の後は breakを記述し、switich 文から離脱させます。
最後に各 case に当てはまらない結果を default で処理します。

◆ 三項演算子

三項演算子とは、if 文のショートカットとして用いられる条件演算子です。
短い記述で条件の場合分けが可能です。

上記の条件式が True の場合は処理 1 が、False の場合は処理 2 が実行されます。

age = 20;

2. 繰り返し文

プログラミングでは「1 回だけ処理を実行」という事は稀で、
「決められた回数」や「条件が続く間」など、処理を繰り返す事が多いです。
この章では、繰り返し文について学習していきましょう。


◆for 文

ループを作る文です。式を 3 つ指定し、その条件が満たされる間、処理が繰り返されます。 

for (式1; 式2; 式3) {
  実行処理;
}

式 1 には変数の宣言(初期化)、式 2 にはループが続く条件、式 3 には変数の処理が記述されます。
式 1 で宣言された変数が式 2 の条件を満たす間、式 3 と実行処理が繰り返されます。
実際にコードを記述して結果を見て行きましょう。 

for (i = 0; i <= 10; i++) {
  console.log(i);
}

◆while 文

while 文は、条件式が True を返す間、処理の実行をループします。

while (条件式) {
  実行処理;
}



コードの例を見てみましょう。

i = 0;
while (i < 6) {
  console.log(i);
  i++;
}

演習 2-2
上記のコードを Runstantで実行し、結果を確認しましょう。

条件が True の間は処理が止まらないため、無限ループに陥りやすいです。
コードを記述する際は注意しましょう。
(例えば上記のコードで i++ の記述を忘れると、無限ループします)



◆do while 文

do while 文は while 文と基本は同じですが、条件式との一致の評価が
最後にあるため、少なくとも 1 回は処理が実行される事になります。

do {
  実行処理;
} while (条件式);

コードの例を見てみましょう。

i = 0;
do {
  console.log(i);
  i++;
} while (i < 6);
演習 2-3
上記のコードを Runstantで実行し、結果を確認しましょう。




while と do while では、条件と違った時の挙動が変わります。
例えば、i の初期値と実行処理を少し変えた次のコードの結果を見てみましょう。 

i = 6;

while (i < 6) {
  console.log("while");
  i++;
}

do {
  console.log("do while");

4. 配列

◆ 配列とは

値に順序をつけ、並べて格納したデータを配列と言います。格納された値を要素、要素の位置を
インデックス(添字)と呼びます。また、配列は変数に格納する事ができます。

上図は変数 fruits に対し、4 つの要素が格納された状態を示しています。fruits 要素のインデックスは、
[] に数値を入れて指定できます。ここで注意するのは、インデックスの1 番目が[0]で表されている点です。
配列の決まり事ですので、「インデックスの初めは 0 から始まる」と覚えましょう。

◆ 配列の作成

配列を変数に格納する際は、主に以下の方法で記述します。

fruits = ["Apple", "Orange", "Banana", "Peach"];

この時、要素が文字列型であれば' ' や " " で囲みます。また要素毎に , で区切る事に注意しましょう。

格納した配列は、インデックス番号を指定する事で出力出来ます。

console.log(fruits[0]);

◆ プロパティ、メソッド、オブジェクト

上記のコード演習では、Array オブジェクトのプロパティとメソッドを学習しました。
ここでは、オブジェクト、プロパティ、メソッドについて、改めて学習しましょう。

● オブジェクト

プロパティの集合体を指し、プログラム上で「モノ」を表現しています。


● プロパティ

名前(キー)と値がセットになったデータを指し、オブジェクトの状態や属性を表します。
値には 文字、数値、論理値、配列、関数を設定できます。


● メソッド

プロパティの一部。「キーと値」の中で、値が関数になっているもので、
特定のデータ型に関連付けられています。




例を挙げて説明します。
「りんご」をオブジェクト(モノ)とした時、そのプロパティ(状態・属性)は次のように表せられます。

category：fruit
color：red
weight:100g
structure：['peel', 'pulp', 'seed', 'juice']

データ上の「りんご」は、種類や色などの情報で表現できるため、
オブジェクトとは「プロパティの集まり」であると言えます。

また、りんごに対して「皮をむく」というメソッドが実行されれば、
「配列 structure から、peel 要素が削除される」 と考える事が出来ます。

● 連想配列

次回の予習となりますが、第 3 回 JavaScript では連想配列を学習します。
これは、プロパティのキーと値を並べた配列で、オブジェクトとして理解しやすいです。

上記の「りんご」の例は、以下の様に記述します。

apple = {
  category: "fruit",
  color: "red",
  weight: "100",
  structure: ["peel", "pulp", "seed", "juice"],
};

console.log(apple["structure"]);

上記のコードは、オブジェクト apple に対し、各プロパティが配列になっている状態です。
連想配列はよく用いられますので、おおよその内容を頭に入れておきましょう。
